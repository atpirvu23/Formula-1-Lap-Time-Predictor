# Formula-1-Lap-Time-Predictor
Formula 1 Lap Time Predictor estimează timpul pe tur folosind date reale F1 (telemetrie) și regresie liniară, rezolvată numeric prin descompunere QR (Householder, Gram–Schmidt).
